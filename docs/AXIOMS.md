# Pointerverse Axioms

## A0 - Local Identity

An object has identity only inside a world and epoch.

## A1 - Relational Existence

An object is not defined by fields alone, but by the relations it participates
in.

## A2 - Typed Directionality

Every pointer is a typed directed relation.

## A3 - Lawful Transformation

A world transition is valid only if produced by a morphism or explicit delta and
admitted by active laws.

## A4 - Observable Projection

The world is never printed directly; observers project measurable views.

## A5 - Trace Preservation

Every committed or rejected transition leaves a replayable trace event.

## A6 - No Unmeasured Myth

Symbolic or inner-world structures must still expose measurable quantities.

## A7 - Drift Boundary

Narrative names must not replace formal constraints.
